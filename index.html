Projet 1

Titre : Site e-commerce pour une marque de vêtements

Description : Nous avons conçu et développé une plateforme e-commerce moderne pour une marque de mode, intégrant des solutions de paiement sécurisées, un système de gestion des stocks en temps réel, et une interface utilisateur intuitive pour améliorer l'expérience d'achat.

Projet 2

Titre : Application mobile pour la gestion des rendez-vous vétérinaires

Description : Développement d'une application mobile permettant aux propriétaires d'animaux de compagnie de prendre facilement des rendez-vous, recevoir des rappels et accéder aux dossiers médicaux de leurs animaux. L'application est compatible avec iOS et Android.

Technologies : Flutter, Firebase, Google Maps API

Projet 3

Titre : Refonte du site web d'une agence de voyage

Description : Nous avons complètement repensé le site web d'une agence de voyage pour améliorer l'expérience utilisateur et augmenter les conversions. Le site inclut désormais une fonctionnalité de réservation en ligne, des guides de voyage interactifs, et une section de blog pour partager des conseils et des expériences de voyage.

Technologies : WordPress, PHP, jQuery
